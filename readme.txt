== Description ==
Use Exchange Rate Widget to display latest Exchange Rate in various currencies. The Exchange Rate are auto updated every 15 minutes

= Customize front-end =

By default the widget looks like all other widgets from the same sidebar you drop it in. It has its own CSS file that overrides the default styles for your list elements. You can use that file to further customize the front-end.
You can also add custom template.


== Installation ==
Upload the Pray Time plugin to your blog, Activate it, then configure your widget from theme wigets.

1, 2, 3: You\'re done!
